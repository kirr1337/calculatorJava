Если вам прям так интересно запустить этот калькулятор (в чем я не сомневаюсь хех) то вам нужно его запускать через командную строку если вы на Windows -> 

Примерно так ->

java --version // Сначала убедитесь что версия джавы выше 21+

java C:\Users\kir\IdeaProjects\calculatorJava\src\main\java\main.java // вот так будет выглядить команда запуска у меня, но вы можете положить main.java в любое место куда хотите (:

например на рабочий стол, тогда ваша директория будет что то вроде C:\Users\ваше-имя-компьютера\Desktop\kir\main.java

Вы также можете написать в github issues если что то не получается, я обязательно помогу! (;
